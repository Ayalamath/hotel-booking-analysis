# Hotel Booking Data Analysis (EDA)

Exploratory data analysis of hotel booking data to study booking behavior, cancellations, and seasonality.

## Dataset
The dataset includes information about:
- hotel type (City / Resort)
- booking status (canceled or successful)
- arrival dates
- length of stay
- number of adults, children, and babies
- room types and customer characteristics

## Analysis Tasks
- Identify top-5 countries by number of successful bookings
- Calculate average stay duration for each hotel type
- Analyze room type mismatches (overbooking cases)
- Study seasonal booking patterns by year
- Compare numeric features (`adults`, `children`, `babies`)
- Create `total_kids` and `has_kids` features
- Calculate churn rate for customers with and without children

## Tools
- Python
- pandas
- matplotlib

## Files
- `hotel_bookings.csv`
